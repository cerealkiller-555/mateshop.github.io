## How to Create Gradients in Canva

**Step 1:** Click on slide background
**Step 2:** Select "Color" → "Gradient"
**Step 3:** Choose "Linear" or "Diagonal"
**Step 4:** Set first color (left/top)
**Step 5:** Set second color (right/bottom)
**Step 6:** Adjust opacity and angle

## Typography

### Fonts to Use
1. **Headlines (H1, H2):** Space Grotesk Bold (or similar geometric sans-serif)
   - Size: 48-60px for slide titles
   - Size: 36-40px for section titles

2. **Body Text:** Inter Regular (or Outfit Regular)
   - Size: 24-28px for main content
   - Size: 18-20px for secondary text

3. **Code/Technical:** Courier New or Fira Code
   - Size: 16-18px for code blocks
   - Color: #4C7249 with light background

### Text Hierarchy
- Title: 54px, Bold, Dark Green
- Subtitle: 32px, Regular, Brand Green
- Body: 26px, Regular, Dark Green
- Small text: 20px, Regular, Gray

## Design Tips & Guidelines

### Consistency
- Use Brand Green (#4C7249) for all accent elements
- Use Dark Green (#2A3626) for all text
- Keep Light Cream (#FDFBF7) as default background
- Use consistent border radius (8-12px) for all boxes

### Visual Hierarchy
- Titles: Largest, bold, Dark Green
- Subtitles: Medium, regular, Brand Green
- Body text: Regular size, Dark Green
- Secondary text: Smaller, gray

### Spacing
- Title to content: 40px gap
- Bullet to bullet: 20px gap
- Card padding: 30px
- Slide margins: 50px from edges

### Images & Icons
- Use professional icons (Noun Project, Flaticon, or built-in Canva icons)
- Color icons in Brand Green when possible
- Keep file sizes optimized
- Use consistent icon style throughout

### Animations (Optional)
- Subtle fade-in for text blocks
- Slide-in for cards from left/right
- Keep animations professional, not distracting
- Use consistent timing (0.5-1 second duration)

### Fonts in Canva
If Space Grotesk unavailable:
- Alternative: Montserrat Bold, Poppins Bold, or Inter Bold
If Inter unavailable:
- Alternative: Roboto, Quicksand, or Outfit

### Quality Checklist
- [ ] All text readable (minimum 18pt)
- [ ] Consistent color scheme throughout
- [ ] Proper alignment and spacing
- [ ] Professional icon usage
- [ ] No spelling errors
- [ ] Clear visual hierarchy
- [ ] Sufficient contrast for readability
- [ ] Consistent font usage
- [ ] All images high quality
- [ ] Transitions are smooth and professional
